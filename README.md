# The Keeper
## A Web Application that tracks all the books you've read, your personal ratings, description, and all the thoughts you've had while reading the book.

---

This Capstone Project is made as a part of the course of *The Complete 2024 Web Development Bootcamp* from Udemy

This project aims to use Node, Express, and PostgresSQL to create a persistive web application which has its own database.

Visit the course here: [The Complete 2024 Web Development Bootcamp](https://www.udemy.com/course/the-complete-web-development-bootcamp/learn/lecture/40325688#overview)

### Tools:
- Node and Express
- PostgresSQL
- EJS
- CSS
